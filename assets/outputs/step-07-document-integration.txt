--- Key Integration Points ---
1. Issue templates provide structured data for Brain parsing
2. Labels drive workflow transitions (needs-triage ÔåÆ triaged ÔåÆ approved ÔåÆ in-progress ÔåÆ done)
3. GitHub Actions bridge GitHub events to SQUAD agent invocations
4. SQUAD workflow YAMLs define the agent pipeline logic
5. Human checkpoints at approval and merge gates
6. agent-issue-mapping.yml routes issues to the correct pipeline
