== Usage Guide ==

### Creating Issues
1. Go to Issues → New Issue
2. Select template: Bug Report, Feature Request, or Squad Task
3. Fill in details — the 'squad' label is auto-applied
4. Submit — Squad triage begins automatically

### Manual Triage
- Add 'squad' label to any existing issue to trigger triage
- Comment '/squad retriage' to re-evaluate assignment

### Monitoring
- Run 'npx @bradygaster/squad-cli triage' locally for real-time triage
- Check squad-heartbeat.yml runs for periodic health checks
- View .squad/agents/{name}/history.md for agent session logs

### Customizing
- Edit .squad/team.md to add/remove agents
- Edit .squad/routing.md to change routing rules
- Edit .squad/agents/{name}/charter.md for agent specialization
- Push changes → sync-squad-labels.yml auto-syncs labels
